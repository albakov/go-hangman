# Игра Hangman написанная на Go

Подробное описание проекта по [ссылке](https://zhukovsd.github.io/java-backend-learning-course/Projects/Hangman/).

## Сборка игры
Команда для сборки:

`make build`

Или:

`go build cmd/main.go && mv main hangman`

## Запуск игры

`./hangman`

# Словарь
Файл со словами:

`config/app.toml`
